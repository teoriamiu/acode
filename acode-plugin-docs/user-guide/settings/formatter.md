# Formatter

Configure default code formatters for each language.

## How It Works

The Formatter settings page lists all supported languages. Tapping on a language opens a selection box where you can choose the default formatter for that language.

Formatters can be:

- **Built-in**: Provided by Acode out of the box.
- **Plugin-based**: Installed via the Plugin Registry (e.g., Prettier).

::: tip
If you don't see your preferred formatter, check the [Plugin Registry](/user-guide/plugins) for available formatter plugins.
:::
