## Overview

The Plugins page allows you to manage all aspects of plugin functionality in Acode:

- View and manage **installed plugins**
- Browse and install new plugins from the **Plugin Registry**
- Manage **purchased plugins**

## Tabs

| Tab           | Description                                                   |
| :------------ | :------------------------------------------------------------ |
| **Installed** | View, update, configure, or uninstall your installed plugins. |
| **Explore**   | Browse and discover new plugins from the registry.            |
| **Purchased** | Manage plugins you have purchased (if applicable).            |

## Plugin Actions

| Action        | Description                                          |
| :------------ | :--------------------------------------------------- |
| **Install**   | Install a plugin from the registry.                  |
| **Update**    | Update an installed plugin to the latest version.    |
| **Uninstall** | Remove a plugin from Acode.                          |
| **Settings**  | Access plugin-specific configuration (if available). |
